Para correr el proyecto en su equipo local
deberá instalar las librerías necesarias mediante los siguientes comandos:

npm install sass
-Para leer archivos codeados en SASS:

npm install react-router-dom
-Propiedades de React Router:

npm config set legacy-peer-deps true
-Esto es necesario para que la instalación de React Reveal se permita en React 18

npm install react-reveal
-Para animaciones y carrusel de imágenes
